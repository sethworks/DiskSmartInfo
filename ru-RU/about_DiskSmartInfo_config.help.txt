КОНФИГУРАЦИОННЫЕ ПАРАМЕТРЫ DISKSMARTINFO

КРАТКОЕ ОПИСАНИЕ

Описание конфигурационных параметров модуля DiskSmartInfo.

ПОЛНОЕ ОПИСАНИЕ

Модуль DiskSmartInfo поддерживает четыре конфигурационных параметра.

Они расположены в файле DiskSmartInfo.config.psd1, находящемся в корневом
каталоге модуля.


SuppressResultsWithEmptySmartData

Этот параметр указывает, должен ли результат выполнения команды включать в себя
диски с пустыми значениями свойства SmartData. По умолчанию такие диски
не отображаются.

Например, если введена следующая команда:

    Get-DiskSmartInfo -AttributeID 100

и среди SMART данных дисков подобный атрибут отсутствует, по умолчанию результат
выполнения команды будет пустым.

Использование параметра -Quiet всегда скрывает такие диски, вне зависимости от
значения данного параметра.

Значение по умолчанию: $true.


TrimDiskDriveModelSuffix

Этот параметр указывает, должна ли команда Get-DiskSmartInfo удалять указывающий
тип диска суффикс из свойства Model объекта Win32_DiskDrive, для того чтобы
его значение соответствовало значениям свойства Model объектов MSFT_Disk
(результат выполнения команды Get-Disk) и MSFT_PhysicalDisk (результат выполнения
команды Get-PhysicalDisk).

Подобными суффиксами являются 'ATA Device' и 'SCSI Disk Device'.

Например, если значение свойства Model объекта Win32_DiskDrive -
'Vendor Model Size ATA Device', то по умолчанию оно будет преобразовано в
'Vendor Model Size'.

Значение по умолчанию: $true.


ReplaceSmartCtlAttributeNames

Этот параметр указывает, что при использовании SmartCtl в качестве источника
данных, возвращаемые имена атрибутов должны быть заменены на имена, определенные
в файлах описания атрибутов: attributes/default.ps1 и attributes/proprietary.ps1.

Значение по умолчанию: $true.


DataHistoryPath

Этот параметр задает каталог, в котором располагаются сохраненные значения атрибутов.

Его значение используется параметрами -UpdateHistory и -ShowHistory команды
Get-DiskSmartInfo.

Если указанный в параметре путь не является полным, то считается, что это путь
относительно корневого каталога модуля.

Значение по умолчанию: 'history'.


ShowUnchangedDataHistory

Этот параметр указывает, должны ли сохраненные значения отображаться для
всех атрибутов.

Если значение параметра $true, то команда Get-DiskSmartInfo выводит все
сохраненные значения.

Если значение параметра $false, то команда выводит только те сохраненные значения,
которые отличаются от текущих.

Этот конфигурационный параметр используется при вызове команды Get-DiskSmartInfo
с параметром -ShowHistory.

Значение по умолчанию: $true.


ArchivePath

Этот параметр задает каталог, в котором хранятся архивные данные.

Его значение используется параметром -Archive команды Get-DiskSmartInfo.

Если указанный в параметре путь не является полным, то считается, что это путь
относительно корневого каталога модуля.

Значение по умолчанию: 'archive'.


Конфигурационные параметры применяются при импорте модуля.
